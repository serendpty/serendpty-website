### `GET` /school
<a id="op-get-school" />

> Return a collection of schools

**Filtering** and **Pagination** are supported for this endpoint provided as **Query Parameters**.

**Filters**
```json
{
  "name_contain":   "<string>",
  "grade_span_id":  "<integer>",
  "composition_id": "<integer>",
  "category_id":    "<integer>",
  "province_id":    "<integer>",
  "district_id":    "<integer>",
  "zone_id":        "<integer>",
  "division_id":    "<integer>"
}
```
_NOTE:_

_Intended usage of **province**, **district**, **zone** and **division**
fields is to better refine data. It follows the following hierarchy._

_**province_id** > **district_id** > **zone_id** > **division_id**_

**IMPORTANT:**

All malformed query parameters will be ignored by the API without informing.
